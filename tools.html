重点工具详情页

这里先整理最常用、最容易被搜索到的工具。每个详情页都会说明它适合解决什么问题、什么时候不适合、和哪些工具搭配更顺手。

AI 助手 ChatGPT

适合写作、编程解释、学习提问和把想法整理成可执行步骤。

 AI 搜索 Perplexity

适合资料调研、找来源线索和快速理解一个陌生主题。

 长文档处理 Claude

适合长文本理解、文档摘要、改稿和内容结构整理。

 开发协作 GitHub

适合代码托管、Issue 管理、Actions 自动化和开源协作。

 Web 文档 MDN Web Docs

适合查 Web 标准、浏览器 API、HTML、CSS 和 JavaScript 概念。

 兼容性检查 Can I use

适合上线前确认 CSS、JS 和 Web API 的浏览器支持情况。

 静态部署 Vercel

适合前端应用、静态站、预览部署和 Git 推送自动上线。

 设计协作 Figma

适合 UI 设计、原型、设计系统和跨角色协作交付。

 轻量设计 Canva

适合快速制作海报、封面、演示文稿和社交媒体图片。

 图片素材 Unsplash

适合寻找摄影图片、封面素材和真实感较强的视觉参考。

 知识管理 Notion

适合搭建个人知识库、项目文档和轻量协作空间。

 翻译润色 DeepL

适合邮件、文档、产品文案和跨语言资料的初步翻译。

 图片压缩 Squoosh

适合在浏览器里压缩图片、转换格式并对比画质。

 数据校验 JSON Formatter

适合格式化接口返回、检查配置文件和临时转换 JSON 数据。
