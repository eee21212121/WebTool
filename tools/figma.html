Design System

Figma 适合做 UI 设计和原型吗

官方地址：figma.com

Figma 适合从线框图到高保真界面的完整设计流程。它的优势是多人协作、组件复用和原型演示，特别适合产品、设计和开发一起确认界面细节。

适合做什么

  • 绘制 UI 界面、交互原型和响应式页面方案。
  • 沉淀组件、颜色、字体和设计系统规则。
  • 让产品、设计和开发在同一文件里评论和确认细节。

使用建议

不要一开始就做过细的视觉稿。早期可以先画结构和流程，确认方向后再补组件、状态和间距。给开发交付时，要标清按钮状态、空状态、错误状态和移动端布局。

注意事项

设计稿不是最终产品。开发前仍要确认真实数据长度、图片比例、按钮文案和极端状态。组件库如果无人维护，很快会变成另一个混乱文件夹。

推荐搭配

Figma 适合高保真设计，Excalidraw 适合早期结构草图，Color Hunt 和 Unsplash 可以补充配色与素材灵感。
