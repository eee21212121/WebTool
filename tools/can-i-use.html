Compatibility

Can I use 怎么检查浏览器兼容性

官方地址：caniuse.com

Can I use 适合在使用新 CSS、JavaScript 或 Web API 前确认浏览器支持情况。它能帮助你判断一个特性是否能直接使用，是否需要降级方案，或者是否应该暂时避开。

适合做什么

  • 上线前检查 CSS Grid、Flexbox、容器查询等布局特性。
  • 确认图片格式、表单能力和浏览器 API 的支持范围。
  • 评估是否需要 polyfill、降级样式或替代方案。

使用建议

不要只看总体支持率，还要看你的用户使用哪些浏览器。如果网站主要面向移动端，需要重点看 iOS Safari 和 Android 浏览器表现。关键布局和表单功能最好在真实设备或浏览器模拟器里再检查一遍。

注意事项

兼容表只说明支持情况，不代表体验一定一致。即使标记为支持，也可能存在细节差异，因此核心页面仍要做视觉检查。

推荐搭配

先用 MDN 理解特性，再用 Can I use 查兼容性，最后在 Vercel 预览部署里检查真实页面。
