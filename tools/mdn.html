MDN Web Docs 怎么查前端文档

官方地址：developer.mozilla.org

MDN Web Docs 适合查 Web 标准、浏览器 API、HTML、CSS 和 JavaScript 概念。遇到前端问题时，先看 MDN 往往比先看零散博客更稳，因为它更关注规范、语法和兼容性。

适合做什么

  • 查询 CSS 属性、HTML 标签和 JavaScript API 的标准用法。
  • 确认示例代码、参数说明和浏览器支持情况。
  • 学习前端基础概念，避免被过期教程误导。

使用建议

查文档时先看语法和示例，再看兼容性表格。如果是新特性，建议配合 Can I use 再确认目标浏览器是否支持。写生产代码前，不要只复制问答社区里的片段，要理解它依赖哪些上下文。

注意事项

MDN 能解释标准和 API，但不会替你判断项目架构。遇到框架问题时，还要回到框架官方文档确认具体写法。

推荐搭配

MDN 查标准，Can I use 查兼容性，Stack Overflow 查具体报错，GitHub Issues 查库本身的问题。
